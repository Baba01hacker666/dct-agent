# DO NOT PUSH TEMPORARY SCRIPTS

Please do not push temporary testing or fixing scripts (such as `patch_*.py`, `setup_*.sh`, `test_*.py`, etc.) to the GitHub repository.

They clutter the repository and are not meant to be part of the main codebase.

If you need to use them, please make sure they are ignored in `.gitignore` or run them locally and delete them before pushing.

Thank you.
